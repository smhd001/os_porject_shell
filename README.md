   - get first words of lines
   ## mrw file 
   - get most repeated word
   ## des file 
   - delete empety spaces
   ## ul file 
   - get uncommented lines

   - get line number
   ## ftl file 
   - get first ten lines

   - change directory

   - exit shell
   and all other shell commands
